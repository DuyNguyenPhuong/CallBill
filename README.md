## :high_brightness: About the project

In countries such as Vietnam, phone bill are calculated based on the minutes you are on the phone. However, after a month, it's hard to check how many minutes have you called. Therefore, I designed this API to gather all the call history and calculated the phone bill each month and notify to users.
